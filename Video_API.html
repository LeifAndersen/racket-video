2.1 Bundled Producers

procedure

(blank length) → producer?

  length : (or/c integer? #f)
Creates a blank producer of length clips.

If length is #f, then the producer generates as many blank frames as its surrounding multitrack requires.

Creates a producer that is a solid color.

Examples:
> (color "green")
> (color "yellow" #:length 10)
> (color 255 255 0)

2.5 Properties

procedure

(producer-length producer) → number?

  producer : (and/c video? converted-video?)
Determines the length of the producer in frames.

Note that this function requires the video to already be converted. In the future, this restriction may be lifted. Also note that this function uses MLT’s length feature, which seems to be a bit buggy at times.

producer is the video who’s length will be tested.
